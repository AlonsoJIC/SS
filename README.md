# SecureSight

Welcome to SecureSight! This project is a modern, user-friendly web application for real-time security analysis of websites. Built with Next.js, React, and Tailwind CSS, SecureSight helps users detect vulnerabilities, misconfigurations, and security risks in seconds.

## Features

- Instant security analysis for any public URL
- Clean, responsive UI with professional design
- Visual security score and technical JSON output
- Recommendations and summary for all users
- Social and portfolio links in the footer

## Tech Stack

- Next.js 14
- React 19
- Tailwind CSS
- TypeScript
- Lucide React Icons
- React Icons

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AlonsoJIC/securesight.git
   cd securesight
   ```
2. **Install dependencies:**
   ```bash
   pnpm install
   # o usa npm install o yarn install si prefieres
   ```
3. **Run the development server:**
   ```bash
   pnpm dev
   # o npm run dev o yarn dev
   ```
4. **Open in your browser:**
   Visit [http://localhost:3000](http://localhost:3000)
